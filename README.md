# AYRA – Your Artificial Companion

AYRA (Artificial Your Responsive Assistant) is a simple AI-based virtual companion designed to interact with users and simulate basic conversations. The project was created as a learning experiment to understand how artificial intelligence assistants work and how interactive systems can be built using Python.

The idea behind AYRA is to explore how a program can receive user input, process it, and respond in a meaningful way. It focuses on building a basic conversational assistant while learning the core concepts of AI and programming.

## Project Overview

AYRA is a Python-based assistant that allows users to interact with a program through text or voice communication. The project was developed to practice programming logic and understand the fundamentals of building AI-driven applications.

It serves as an introductory project for exploring conversational AI systems and improving problem-solving skills in Python.

## Features

* Basic conversational interaction
* User input processing
* Simple response generation
* Beginner-friendly AI assistant

## Technologies Used

* Python
* Speech Recognition
* Text-to-Speech (TTS)
* Basic AI assistant logic

## Requirements

To run this project, make sure the following are installed on your system:

* Python 3.8 or higher
* pip (Python package manager)

Python libraries used in this project:

* speechrecognition
* pyttsx3
* pyaudio
* openai

These libraries are used for speech recognition, voice output, and generating AI-based responses.

## Installation

Clone the repository:

git clone https://github.com/Nitink420/AYRA-your-artificial-companion.git

Move to the project directory:

cd AYRA-your-artificial-companion

Install the required libraries:

pip install speechrecognition pyttsx3 pyaudio openai

Run the program:

python main.py
